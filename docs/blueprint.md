# **App Name**: Personal Canvas

## Core Features:

- Hero Section: Display a prominent introduction with a title, subtitle, and a call-to-action button to showcase projects.
- About Section: Present a personal overview with text and an image, highlighting skills and experience.
- Projects Grid: Showcase a collection of projects with images, titles, descriptions, and technology tags.
- Contact Form: Provide a functional form for users to send messages, including validation and an alternative contact method.
- Footer: Include minimal links to CV, social networks, and copyright information.
- Optimized Image Loading: Ensure efficient loading and display of project images using `next/image`.
- Downloadable CV: Provide a link to download the CV as a static PDF file from the /public directory.

## Style Guidelines:

- Primary color: Moderate blue (#3B82F6) for a professional and techy feel.
- Background color: Very light blue (#EBF4FF) for a clean, minimalist base.
- Accent color: Light blue (#60A5FA) for interactive elements and highlights, creating subtle contrast.
- Body and headline font: 'Inter' sans-serif for a modern, readable style. Note: currently only Google Fonts are supported.
- Use simple, geometric icons to represent technologies and project categories.
- Implement a responsive layout with breakpoints for mobile, tablet, and desktop.
- Incorporate subtle hover effects on project cards to enhance interactivity.
- Header: height 80px, padding horizontal 32px (desktop); 70px, padding 24px (tablet); 60px, padding 16px (mobile).
- Hero main text font size: 3rem (~48px) desktop, 2.5rem (~40px) tablet, 2rem (~32px) mobile.
- Hero tagline font size: 1.5rem (~24px) desktop, 1.25rem (~20px) tablet, 1rem (~16px) mobile.
- About section: max width 960px, margin auto (desktop); 720px (tablet); 90% viewport width (mobile).
- Projects grid: 3 columns with 32px gap (desktop); 2 columns with 24px gap (tablet); 1 column with 16px gap (mobile).
- Project card: width 100%, height auto, 24px padding (desktop); 20px padding (tablet); 16px padding (mobile).
- CTA button: 16px 32px padding, 1rem font size (desktop); 14px 28px (tablet); 12px 24px (mobile).
- Contact form width: 600px (desktop); 480px (tablet); 90% mobile width (mobile).
- Spacing between sections: 96px (desktop), 64px (tablet), 48px (mobile).
- Background: #FFFFFF, Text: #333333, Subtitles/Lines: #666666, Links/Buttons: #2563EB, Button Hover: #1E40AF.
- Main header font: modern sans-serif, body font: same family, base size 16px, line-height 1.5.
